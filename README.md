# Barometr

**Observatoire mondial des équilibres socio-économiques**

[barometr.vercel.app](https://barometr.vercel.app)

Carte interactive mondiale couvrant 14 domaines socio-économiques (santé, éducation, écologie, sécurité, numérique, égalité…) sur 62 pays, à partir de données publiques World Bank, WHO, OECD et IMF.

## Stack

- **Next.js 14** — App Router, export statique
- **MapLibre GL** — carte vectorielle interactive
- **Tailwind CSS** — design mobile-first
- **Vercel** — déploiement continu

## Démarrage

```bash
npm install
npm run dev
```
